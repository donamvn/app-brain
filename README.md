## 🚀 Setup Custom Domain

### Bước 1: Cấu hình DNS tại donam.vn

Thêm CNAME record:
```
Type: CNAME
Name: tuduyso
Value: [your-github-username].github.io
```

### Bước 2: Enable GitHub Pages

1. Vào Settings → Pages
2. Source: Deploy from a branch
3. Branch: main → /docs
4. Custom domain: tuduyso.donam.vn
5. ✅ Enforce HTTPS
